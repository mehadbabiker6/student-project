
# محلل النصوص البسيط

## (الوصف)
هذا المشروع عبارة عن أداة برمجية بسيطة بلغة Python تقوم بتحليل النصوص المدخلة، حيث تحسب عدد الكلمات، عدد الأحرف، وتكتشف تكرار الكلمات الأكثر شيوعاً. الهدف منه هو مساعدة الكتاب والمطورين على فهم هيكلية نصوصهم بسرعة.

##  (تعليمات التثبيت)
لتشغيل المشروع على جهازك، اتبع الخطوات التالية:
1. تأكد من تثبيت لغة Python على جهازك.
2. قم بتحميل ملفات المشروع:
   `git clone https://github.com/mehadbabiker6/student-project.git`
3. انتقل إلى مجلد المشروع ثم قم بتشغيل الملف الأساسي:
   `python analyzer.py`

##  (مثال على الاستخدام)
عند تشغيل البرنامج، قم بإدخال النص الذي تريد تحليله:
- **Input:** "Hello world, hello GitHub"
- **Output:** - Word Count: 4
  - Character Count: 25
  - Most Common Word: "hello"
